# Docker usage guide

## Quick Start



### Development Environment

```bash
npm run docker:dev
```

### 

### Production Environment

```bash
npm run docker:prod
```

### 

### Build

```bash
npm run docker:build
```

### 

### Stop Services

```bash
npm run docker:down
```
